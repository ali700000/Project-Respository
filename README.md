# Project Title

Time of Exploration

## Project Description
What did you make?

My project is a game menu inspired by modern RPG games. The interface is a game menu that features a nagivation menu and a layout that is easy for the players to understand.

## Intended User or Player
Who is this interface designed for?

This interface is designed for players who enjoy adventure RPG games. The audience includes players who like immersive visuals and simple navigation in modern games.

## User / Player Goal
What should the user or player be able to do or understand?

The goal of this interface is for players to engage with the main menu and understand the game's themes and how to nagivate the important options.

List the inspiration interfaces you used and briefly explain what you learned from them.

## Design Choices
- Layout: The interface has a header, navigation bar, cards, information sections, and footer.
- Color: The interface uses a black and dark gray color palette with a bit of yellow to make the interface visually appealing while also maintaining readability.
- Typography: The large headings in the interface establish typography and improve readibility on desktops and mobile devices.
- Visual hierarchy: The main heading draws the player's attention. Important action like the "Begin Adventure" button are highlighted, followed by the cards that showcase additional infomation.
- Feedback states: Buttons, navigation links, and cards all include hover effects to provide visual feedback for players.
- Responsive design: The layout of the interface adapts to mobile devices and adjusts spacing to improve readibility for players playing on mobile devices.
